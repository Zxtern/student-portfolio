# Student Portfolio

## Overview

This project is a personal student portfolio built using React. The purpose of this application is to present my academic background, coursework, and development experience in a structured and accessible format. The portfolio serves as a centralized location where visitors can learn more about my technical skills, projects, and interests within computer science.

The site is designed as a simple single-page application that organizes information across several sections including a home page, background information, and coursework. The goal of the project is not only to showcase my work, but also to demonstrate an understanding of frontend development practices such as component-based architecture, modular styling, and organized project structure.

## Technologies Used

* React (JavaScript library for building user interfaces)
* JavaScript (ES6+)
* HTML5
* CSS3
* Node.js / npm
* Git and GitHub for version control

### Directory Explanation

**assets/**
Contains static resources such as images used throughout the site.

**components/**
Contains reusable interface components used across multiple pages. For example, the navigation bar is implemented as a reusable component.

**pages/**
Contains the main sections of the portfolio application. Each page represents a distinct section of the website such as the homepage, personal background, or coursework.

**App.js**
Acts as the main application component that combines the navigation and individual pages together.

**index.js**
The entry point of the React application where the React application is mounted to the DOM.

This portfolio was created as part of my journey in computer science to document my learning progress and development experience. It demonstrates my ability to structure a frontend application using React and maintain an organized project architecture.

## Running the Project

To run this project locally:

1. Clone the repository
2. Install dependencies

```
npm install
```

3. Start the development server

```
npm start
```

The application will run locally at:

```
http://localhost:3000
```
